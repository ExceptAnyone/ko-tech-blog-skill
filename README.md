# Korean Tech Blog Writing Skill

A Claude Code skill for writing Korean tech blog posts in [Toss Tech Blog](https://toss.tech) style.

### Features
- 7-step writing process
- 3 templates (Problem-solving, Concept-explaining, Basic)
- Toss style guidelines
- Automatic YAML frontmatter generation
- Code examples and image placement suggestions

### Installation
```bash
/plugin marketplace add ExceptAnyone/ko-tech-blog-skill
/plugin install ko-tech-blog
```

### Usage
```
Write a blog post about my Next.js optimization experience in Toss style.
```
